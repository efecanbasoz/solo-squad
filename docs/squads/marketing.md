# Marketing Squad

Everything a digital marketing team runs day-to-day. Built for agencies that work across SEO, paid, social, content, and tracking.

**Plugin name:** `marketing-squad`

## Skills

| Skill | What it does |
|-------|--------------|
| `/seo-audit` | Technical SEO audit: crawlability, Core Web Vitals, on-page factors, content gaps, backlink profile. Priority matrix output. |
| `/ppc-strategy` | PPC campaign strategy for Google, Meta, LinkedIn, TikTok. Account architecture, bidding, audiences, budget allocation. |
| `/social-strategy` | Cross-platform social strategy: platform selection, content pillars, posting cadence, community playbook. |
| `/content-plan` | Editorial calendar: keyword-mapped content briefs, publishing schedule, distribution plan. |
| `/ad-creative` | RSA headlines, Meta copy, Performance Max assets, A/B test plans, creative refresh cycles. |
| `/tracking-audit` | GTM container review, GA4 config, conversion tracking verification, CAPI setup, attribution analysis. |
| `/paid-audit` | 200+ point paid media account audit. Scores structure, targeting, bidding, creative, tracking, competitive. |
| `/ai-citation` | Audit brand visibility in ChatGPT, Claude, Gemini, Perplexity. AEO/GEO optimization plan. |

## Commands

| Command | What it does |
|---------|-------------|
| `/client-audit` | Full marketing audit: SEO + PPC + tracking + social + AI citation in one combined report |
| `/monthly-report` | Cross-channel monthly performance report with trends and recommendations |
| `/campaign-launch` | Pre-launch checklist across all channels with go/no-go gate |

## Agents

| Agent | Role |
|-------|------|
| SEO Analyst | Technical SEO specialist. Crawl budgets, search intent, topical authority. |
| Media Buyer | Paid media across platforms. ROAS-focused, questions vanity metrics. |
| Content Strategist | Editorial strategy. Decides what to create and why, not how to write it. |

## Hooks

After every file write, a hook reminds you to verify UTM parameters and conversion tracking if the file looks like a landing page.

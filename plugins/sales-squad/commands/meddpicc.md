# MEDDPICC Quick Score

Fast deal qualification. Score each MEDDPICC element, identify the top gaps, get an action plan.

## Process

1. **Gather context.** Ask the user about the deal. If they provide a deal name or description, work with that. If not, prompt for: company name, deal size, current stage, main contact.

2. **Score each element.** Walk through each element and ask the user to rate 1-5 (or provide enough context for you to rate):

   | Element | Question | Score |
   |---------|----------|-------|
   | **Metrics** | Can the buyer quantify the value of solving this? | /5 |
   | **Economic Buyer** | Have you accessed the person who signs the check? | /5 |
   | **Decision Criteria** | Do you know exactly how they'll evaluate options? | /5 |
   | **Decision Process** | Do you know every step from here to signed contract? | /5 |
   | **Paper Process** | Is legal/procurement/security review mapped? | /5 |
   | **Implied Pain** | Is there urgency? What happens if they do nothing? | /5 |
   | **Competition** | Do you know who else is in the deal? | /5 |
   | **Champion** | Do you have someone inside actively selling for you? | /5 |

3. **Calculate composite.** Total out of 40.
   - 32-40: Strong — focus on execution and timeline
   - 24-31: Promising — address gaps before advancing
   - 16-23: At risk — critical gaps need immediate action
   - Below 16: Unqualified — have a hard conversation about investing time

4. **Identify top 3 gaps.** The lowest-scoring elements. For each gap:
   - Why it's dangerous at this stage
   - Specific action to strengthen it
   - Who should do it and by when

5. **Output.** One-page deal scorecard with scores, gaps, and action plan.

## Rules

- This is a 5-minute exercise. Don't over-analyze.
- Be honest about deal health. Hope is not a strategy.
- Different from `/deal-strategy` which is comprehensive — this is a focused scoring tool.
- If the deal scores below 16, recommend qualifying out or pausing investment.

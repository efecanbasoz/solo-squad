Morning standup generator. Review recent git activity, open PRs, and project status across all active work.

For each active project/client:
- What shipped yesterday (commits, merged PRs)
- What's in progress (open branches, draft PRs)
- Blockers (stale PRs, failing CI, waiting on input)

Format as a quick-scan list. Total should fit in one screen.
Highlight anything that needs attention TODAY in bold.
